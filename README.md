# Explainable Quantum Kernel SVM for Heart Disease Prediction Using SHAP

This repository contains the code, data processing pipeline, and experimental
results for the study "Explainable Quantum Kernel SVM for Heart Disease
Prediction Using SHAP." The project investigates whether a fidelity-based
Quantum Support Vector Machine (QSVM) can serve as a competitive and
explainable alternative to a classical SVM for binary heart-disease
prediction, using SHAP to interpret both models in the original clinical
feature space.
